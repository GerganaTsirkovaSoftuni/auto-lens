# Auto Lens Website

Premium multi-page website for **Auto Lens** in Bulgarian language.

## Pages

- Home
- About
- Services
- Contact
- FAQ
- Privacy Policy
- Terms of Service
- Cookie Policy

## Tech

- HTML5
- CSS3 (dark theme + glassmorphism + responsive grid)
- Vanilla JavaScript (mobile nav, scroll reveal animations, portfolio popup/lightbox, show-more gallery)
- SVG branding assets (logo, favicon, custom icon set)

## Run locally

Open `index.html` in a browser, or serve with any static server.

Example with Node.js:

```bash
npx serve .
```

## Accessibility highlights

- Semantic landmarks and headings
- Skip link
- Keyboard-friendly navigation and interactive controls
- Keyboard-friendly gallery popup with Escape close
- `prefers-reduced-motion` respected
